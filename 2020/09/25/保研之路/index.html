重要时刻 と 心路历程








这半年就是大起大落大起大落大落大起(啪)
有激动有惆怅有绝望，还好最后的结果不算太差(ﾉ∀`*)


特殊的一年
今年有疫情，有政局变化，应该是保研最难的一年 (我们院的保研情况却意外的不错  


特殊情况有：


    

  1. 夏令营基本转为线上，省去了不少报名的麻烦，不用到处跑，校方也花不了多少钱(结果大家都海起来了
    2. 

  2. 出国形式不乐观，不少出国大佬选择保研卷起来卷起来
    3. 

  3. 推免政策大变动！sum改成max，论文要求更严培养六边形战士
    4. 



所以今年的经验仅供参考哦~


一些建议
准备
    

  1. 尽早开始准备。我由于复习启动的过晚所以没敢申清北的数学直博……（他们出预推免也太早了
    2. 

  2. 多搜集信息。各个院校都会有预推免、夏令营的通知，可以查看往年的先准备着。有些保研机构的汇总贴也可以收藏一下。
    3. 

  3. 尽管保研加分对论文和竞赛做了限制，但是在外校的面试中还是很看重这些的，所以如果能有一两项出彩的会比较好，面试时也有话题聊。但这些不用强求，能力强的话自然而然就会有，不建议为了保研而去做。（别卷了别卷了
    4. 

  4. 科研经历没有论文还可以打磨一下自己的SRTP项目。我就是靠着机器学习求解微分方程的项目拿到清华的优营的吧。
    5. 



申请
    

  1. 定好目标院校和专业。这里可以设几档志愿学校，每档学校都有申请，最后能拿到offer了再挑。
    2. 

  2. 可以主动联系导师发自荐信，网上都有相关的建议。但好像我这一圈下来感觉没什么必要，夏令营会有一对一交流，如果是导师单独招生的话导师会主动邀你面试的。复旦和港中文联系到老师了也没过院里的面试
    3. 

  3. 申直博的话要做好吃苦的准备，要热爱科研。（可能我现在还没做好吧
    4. 

  4. 保研的学生也可以考虑试试出国或香港的项目哦~ 比如港中文和港科大的early admission和summer school对保研人就很友好，不需要当时有雅思托福成绩，拿到con offer后再考就行！
    5. 

  5. 可以同时准备申请多个专业，但跨专业要补的很多，如果之前没有积累的话还是瞄准一个努力补专业知识比较好。
    6. 



专业
    

  1. 数学跨专业是相对容易的。计算机，金融和大数据方向都是很欢迎数学同学的，甚至我跨的地学也是，所以可以选的路很多，看你喜欢哪一个。
    2. 

  2. 跨计算机至少需要熟练算法和编程，可以去各大OJ刷题。大一对编程很感兴趣或者有信息竞赛经历的建议加入校CPC俱乐部，后面可以打打ACM竞赛。
    3. 

  3. 交叉专业也可以多考虑一下。比如生物统计，北叉清茶（这个难度很大），清华地球系统科学系（好多方向是地学叉计算机叉数学的，所以我会有竞争优势）
    4. 



夏令营确定录取
    

  1. 老师/学校给口头offer后一定要确认清楚，是不是自己一定能去。有的学校超额发offer到最后却会鸽人的情况也是有的。
    

    2. 

  2. 鸽其他学校/老师一定是越早越好！确定最终选择offer的话请马上发拒信通知给不去的学校，给老师补招和其他学生机会。如果达成了口头/书面承诺再鸽，属于失信行为，可能会受到制裁以及影响学弟学妹报考。
    

    3. 



暂时想到这么多啦，有什么要问的可以在评论区提出来哦~
