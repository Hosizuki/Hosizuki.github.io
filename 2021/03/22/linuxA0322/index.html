    Ch6 Linux 文件与目录管理

    常见处理目录命令

    • cd dirname 变换目录，cd == cd ~都是回到家目录，cd -是回到刚才的目录。
    • pwd 显示当前目录，-P可以显示无链接的实际路径。
    • mkdir newdir 建立新目录，-p d1/d2/d3建立多层目录，-m 711设定权限。
    • rmdir olddir 删除目录，-p删除多层空目录。
    • ls 查看目录文件，一般ll == ls -al为所有文件及信息，-h显示可读文件大小。

    另外，.或./代表当前目录，..是上一层哦。

    复制文件

    cp （是 copy 不是 couple

    • -a == -d + -r + --preserve=all == link 复制链接 + 递归复制 + 保持属性复制
    • -s 创建快捷方式（符号链接）
    • -i 询问是否覆盖
    • -l 硬式链接
    • -u update，时间更新才复制
    • -f 强制复制，若目标已存在且无法开启，则移除后再试一次
    • cp source1 source2 ... dir 多个文件复制，只能复制到目录下

    删除文件

    rm remove

    • -i 询问是否删除，root 默认
    • -r 递归删除
    • -f 强制删除，不询问
      使用通配符*来批量删除，如rm *.txt
      使用\rm来忽略alias 的指定选项。

    移动文件

    mv -fiu 与复制的选项一致。也可以多文件一起移动到某个目录下。

    取得文件名和路径

    对于绝对路径，使用basename和dirname分别取得文件名和文件路径。

    查看文件内容

    • cat 由第一行开始显示文件内容 -n显示行号，-T用^T显示Tab键，-v列出看不见的特殊字符，-E显示结尾的断行字符$，-A==-vET
    • tac 从最后一行倒着显示（还是 cat 的倒着写
    • nl 显示的时候输出行号！
    • less 与 more 类似，但是比 more 更好的是，他可以往前翻页！
    • head 只看头几行
    • tail 只看尾巴几行
    • od 以二进制的方式读取文件内容

    创建文件与修改文件时间

    touch 后接不存在的文件==创建该文件，存在文件则修改 access time 和 modification time 为当前。

    预设权限，隐藏属性与特殊的文件权限

    与umask，chattr，S*ID有关，先略过。

    观察文件类型

    file 可以简单的先判断这个文件的格式为何，比如 ASCII text，data等等。

    查找指令与文件

    which [-a] command -a显示所有由 PATH 目录中可以找到的指令
    whereis [-bmsu] 文件或目或目录名

    • -l :列出 whereis 会去查询的几个主要目录
    • -b :只找 binary 格式的文件
    • -m :只找在说明文件 manual 路径下的文件
    • -s :只找 source 来源文件
    • -u :搜寻不在上述三个项目当中的其他特殊文件

    locate [-ir] keyword

    • -i ：忽略大小写的差异；
    • -c ：不输出档名，仅计算找到的文件数量
    • -l ：仅输出几行的意思，例如输出五行则是 -l 5
    • -S ：输出 locate 所使用的数据库文件的相关信息，包括该数据库纪录的文件/目录数量等
    • -r ：后面可接正规表示法的显示方式

    find [PATH] [option] [action]

    1. 与时间有关的选项：共有 -atime, -ctime 与 -mtime ，以 -mtime 说明
      • -mtime n ：n 为数字，意义为在 n 天之前的”一天之内”被更动过内容的文件；
      • -mtime +n ：列出在 n 天之前(不含 n 天本身)被更动过内容的文件档名；
      • -mtime -n ：列出在 n 天之内(含 n 天本身)被更动过内容的文件档名。
      • -newer file ：file 为一个存在的文件，列出比 file 还要新的文件档名
    2. 与使用者或组名有关的参数：
      • -uid n ：n 为数字，这个数字是用户的账号 ID，亦即 UID ，这个 UID 是记录在 /etc/passwd 里面与账号名称对应的数字。
      • -gid n ：n 为数字，这个数字是组名的 ID，亦即 GID，这个 GID 记录在 /etc/group
      • -user name ：name 为使用者账号名称喔！例如 sinki ；
      • -group name：name 为组名喔，例如 users ；
      • -nouser ：寻找文件的拥有者不存在 /etc/passwd 的文件。
      • -nogroup ：寻找文件的拥有群组不存在于 /etc/group 的文件。当自行安装软件时，很可能该软件的属性当中并没有文件拥有者，在这个时候，就可以使用 -nouser 与 -nogroup 搜寻。
    3. 其他
      • -name filename：搜寻文件名为 filename 的文件，可以使用通配符*；
      • -size [+-]SIZE：搜寻比 SIZE 还要大(+)或小(-)的文件。这个 SIZE 的规格有： c: 代表 byte， k: 代表 1024bytes。
      • -type TYPE ：搜寻文件的类型为 TYPE 的，类型主要有：一般正规文件 (f), 装置文件 (b, c), 目录 (d), 连结档 (l), socket (s), 及 FIFO (p) 等属性。
      • -exec command ：command 为其他指令，-exec 后面可再接额外的指令来处理搜寻到的结果，比如ls -l，但不能用别名ll。
