记录一下网站的进化史~
我会把一些部件的实现方式记录在这哦。坑都踩好了

2022年

4月10日-14日

  • 内容分区：由主站分离出学习记录内容(S站)并在原域名重新搭建。
  • S站配置:
    • ./source/js/_app/global.js 里 const visibilityListener = function () { 注释掉。修改的部分加了署名注释。
    • 在 ./layout/_partials/footer.njk 中插入
        {%- if theme.footer.count %}
  <span class="post-meta-divider">|</span>
  <span class="post-meta-item-icon">
  <i class="ic i-eye"></i>
  </span>
  <script async src="//busuanzi.ibruce.info/busuanzi/2.3/busuanzi.pure.mini.js"></script>
  <span id="busuanzi_container_site_uv" > <span id="busuanzi_value_site_uv"></span>  </span>
  <!-- 不蒜子计数 -->
      并注释掉 @ {{ alternate if alternate else title}}
    • 删除 copyright；关掉页面 cache，双语界面国际化；LeanCloud 评论区（参考）。

4月8日

  • 网站迁移：需要修改_config.yml里的 deploy，包括 git@hosi.github.com，顺便也改了.git/config和git config --global user.email。修改 disqusjs 的域名、api的域名，以及 Migration Tools。
  • 访问次数重置…… 1594人次

2021年

6月3日

  • 实装搜索功能！在搜索栏中搜索”搜索”找到文章，看看具体改了哪些吧！
  • 修改主题色。具体操作同样在那篇文章里。
  • 修复音乐播放器的样式，直接使用aplayer播放。

3月16日

  • 更新nexmoe主题，添加归档。更新后需重新恢复计数器、看板娘、置顶标签。其实是发现样式崩了才爬过去更新
  • 更换网站标签图标和个人头像，添加联系方式组件（在主题的配置文件中修改）。
  • 删除回到顶部按钮：注释\layout\layout.ejs里含有回到顶部的一行。（因为挡住咱的看板娘了！人又菜不懂怎么改位置
  • 取消复制文本时追加的著作权信息，作为小白找的好苦…… 删除\nexmoe\layout\_partial\after-footer.ejs里的
      <% if (theme.cdn.copy) { %>
  <script src="<%- theme.cdn.copy %>"></script>
  <% } %>
    虽然成功了但仍然不懂这个theme.cdn.copy脚本是怎么把某github.io的copy.js导进来的……
  • 找不到此网站问题：原因是运营商DNS污染（域名指往不正确的IP地址），可以通过网络和Internet设置->更改适配器选项->网络属性->Internet协议版本4（TCP/IPv4）属性->使用下面的DNS服务器地址->223.5.5.5（阿里）或8.8.8.8（谷歌）解决该问题。

3月1日

  • 按Github要求配置Token authentication：配置SSH登陆。命令我用的ssh-keygen -t rsa -b 4096（其实要不是突然不能更新网站了我才不配置……）
  • 修复tag大小写问题：\.deploy_git\.git\config的 ignorecase=true 改为 ignorecase=false。

2020年

12月22日

  • Mathjax可能会出问题不显示，需要在git bash中执行hexo clean。
  • 封面图片用相对网站位置，如/pictures/miku.png，方便管理且能够避免raw.githubusercontent.com抽风。

11月1日

  • 恢复Marked渲染，折腾好久：解决Mathjax与代码高亮的冲突。下划线可以加反斜杠或者用{% raw %}{% endraw %}把latex包起来。

10月23日

  • 支持$LaTeX$:使用 Mathjax。 好像只能强制全局渲染，关不掉了…… 不使用Kramed渲染。

10月5日

  • 图片加载问题：解决Github网页上图片显示失败的问题。需要各位手动设置……（反正我现在访问没啥问题了

10月4日

  • 网站主页：目前是仿着11岁时的网站搭的页面~ 使用了主题里的py模板。
  • 置顶文章：参考hexo博客优化之文章置顶+置顶标签。不过这篇是next主题的，Nexmoe需要在\layout\_partial\_post\meta.ejs里的<div class="nexmoe-post-meta nexmoe-rainbow"> 和 \layout\index.ejs里的<div class="nexmoe-post-meta nexmoe-rainbow">中插入
      <% if (page.top){ %>
     <a> <i class="nexmoefont icon-right"></i>
      置顶</a>
  <% } %>

9月28日

  • 网站访问统计：不蒜子计数 这个主题的话可以把代码分别插在\\layout\\_partial\\sidebar.ejs，\\layout\\_partial\\copyright.ejs和\\layout\\layout.ejs里。
  • 文章加密：hexo-blog-encrypt 加密后有些部件失效了就，包括音乐播放、导航。脑壳痛
  • 看板娘：教程 两行代码<script src="/live2d-widget/autoload.js"></script> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/font-awesome/css/font-awesome.min.css">同样插在\\layout\\layout.ejs里。 位置在waifu.css里调，left改成right

9月27日

  • 个人信息界面
    图片并排实现：查了不少方法都不行，最后发现直接用表格再插图就好（有够偷懒
    隐藏标签：使用 <details> 元素 貌似IE不支持
    封面图片：这东西用相对路径的话主页就不显示图片，于是直接用了在上传网站后Github里存放图片的网络位置。

9月26日

  • 开始搭建博客

    1. 建站教程：用 Hexo 和 GitHub Pages 搭建博客 另外还可参考如何用github page搭建博客？
    2. Hexo文档：文档 | Hexo
    3. 网站模板：Nexmoe’s Theme 唔分类栏好像有点问题……
    4. 主题开发者的blog：折影轻梦 (以后有钱了我也要买个域名
    5. 博客编写：
      • Markdown教程 (太菜了啥都不会
      • Sublime Markdown插件安装
    6. 一些命令：
       $ hexo new page "新文章"
 $ hexo clean
 $ hexo g
 $ hexo g -d
 $ hexo s
      这里 $ hexo d 有可能会报错Please tell me who you are，解决办法

  • 评论系统
    这里我用的是Disqus（需科学），参考Nexmoe 评论系统的Disqusjs，apikey是配置 Disqus Application 时获取的公钥。

  • 赞助页面 觉得有用的话就赏点吧 孩子可穷了
    本来没想搞的，炜神建议搞一个
    插入音频：hexo-tag-aplayer
